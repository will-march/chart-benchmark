Chart Library Benchmark

Measures synchronous render time per operation. Canvas 2D calls (Lightweight Charts by TradingView, Apache ECharts) are synchronous JS — timed with performance.now(). WebGL calls (Finterm-style renderer) are async GPU work — timed with gl.finish() which blocks until the GPU completes. Each test: 500 pan steps + 200 zoom steps. Reports p50 / p95 / max in ms. Effective FPS = min(60, 1000 / p50).
